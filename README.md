# High-Fidelity Local Music Player (HiFi-Player)

HiFi-Player is a premium, high-performance Android application designed for audiophiles who want to experience their local music downloads in the highest possible quality. 

## Key Features
- **Lossless Support**: Play FLAC, WAV, and ALAC files at their native bitrates.
- **Material 3 Design**: A modern, fluid interface with glassmorphism and dynamic color support.
- **Smart Folder Discovery**: Automatically scans your Downloads and Music folders for new tracks.
- **Gapless Playback**: Seamless transitions between tracks.
- **Audio Engine Control**: Advanced equalizer and sample-rate switching for DAC users.

## Software & Technologies Used
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Media Engine**: Jetpack Media3 (ExoPlayer)
- **Dependency Injection**: Hilt
- **Database**: Room (for library management)
- **Image Loading**: Coil (optimized for album art)
- **Architecture**: MVVM (Model-View-ViewModel)

## Getting Started
1. Clone the repository.
2. Open in Android Studio (Ladybug or later).
3. Sync Gradle and run on an Android device (API 26+).
